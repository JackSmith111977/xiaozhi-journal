**经验教训:**
- `role="radio"` 不支持 `aria-pressed`，仅支持 `aria-checked`。`aria-pressed` 用于 `role="button"` 或 `role="tab"`
- Story 建议注入 `class="warm-sun"/"starry-night"`，但项目 Tailwind 使用 `@custom-variant dark (&:is(.dark *))`，实际需要注入 `class="dark"` 才能触发 `.dark` 色板。Story 建议仅供参考，必须根据实际 Tailwind 配置调整

- hex → oklch 转换需近似计算，无法精确匹配。使用 `oklch(L C H)` 近似即可，Tailwind 渲染结果肉眼一致
